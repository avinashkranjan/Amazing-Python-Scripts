<h1 align="center">URL Shortner GUI</h1>
GUI based It shortens your given website URL.

---------------------------------------------------------------------

## Modules Used
- pyshorteners
- clipboard
- tkinter

## How it works
- It takes the given your website URL as input. 

- It shortens the given URL.

- It Returns the shortened URL in the output.
